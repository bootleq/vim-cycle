Cycle.vim
=========

Cycle text to predefined candidates.

  - `true` =&gt; `false`
  - `No` =&gt; `Yes`
  - `是` =&gt; `否`
  - `"` =&gt; `'`
  - `Rails Metal` =&gt; `Thrash` =&gt; `Technical Death`
  - `<em>foo</em>` =&gt; `<strong>foo</strong>`
